# Shell Enfocado en el Aprendizaje de Sistemas Operativos

## Instrucciones (desde el root del proyecto)

- **Crear el CMake del proyecto:**
```
cmake .
```

- Compilar el programa:

```
cmake --build .
```

- Ejecutar el programa:

```
./yuniks
```
